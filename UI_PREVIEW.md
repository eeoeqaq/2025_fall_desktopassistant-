## 特殊功能说明

### 进度条动画
进度条会实时更新，显示当前进度百分比：
```
[██████████████████████████████░░░░░░░░░░] 75%
```

### 键盘快捷键
在专注模式下：
- 按 `p` 或 `P` - 暂停/继续
- 按 `i` 或 `I` - 记录干扰
- 按 `q` 或 `Q` - 放弃番茄钟（需要确认）

### 音效提示（Windows）
- 完成番茄钟：三音递增（哔哔哔～）
- 解锁成就：三音递增（哔哔哔～）

### 休息提示
- 短休息（5分钟）：每完成一个番茄钟后
- 长休息（15分钟）：每完成4个番茄钟后
- 可按任意键跳过休息

## 设计特点

1. **视觉层次清晰** - 使用 Unicode 框线字符（═ ─）创建清晰边界
2. **表情符号丰富** - 使用 emoji 增强可读性和趣味性
3. **实时反馈** - 进度条、倒计时实时更新
4. **一致性** - 与现有模块 UI 风格保持一致
5. **响应式** - 所有操作都有明确的视觉/听觉反馈
