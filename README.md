# 2025_fall_desktopassistant

桌面助手 - 一个多功能的命令行助手工具

## 功能模块

### 1. 账本
财务管理和记账功能

### 2. 备忘录
- 记事本
- 密码保险箱
- 日记本

### 3. TodoList
任务管理和待办事项

### 4. 多功能计算器
表达式计算器

### 5. 24点游戏
数学益智游戏

### 6. 🍅 番茄钟专注系统 (NEW!)
完整的番茄钟工作法实现，包含：
- 25分钟专注计时器（可自定义）
- 任务管理和追踪
- 干扰记录功能
- 专注统计和7天趋势
- 成就系统（7个等级）
- 历史记录查看
- 自定义设置
- 报告导出

## 构建说明

### Windows
```bash
mkdir build
cd build
cmake ..
cmake --build .
```

### Linux (部分功能受限)
```bash
mkdir build
cd build
cmake ..
make
```

注意：番茄钟系统的键盘快捷键和音效功能仅在 Windows 平台可用。

## 使用说明

运行程序后，选择相应的功能模块即可使用。

### 番茄钟使用指南

1. 从主菜单选择 "6. 🍅 番茄钟专注系统"
2. 选择 "1. 🍅 开始番茄钟" 启动计时
3. 在专注过程中：
   - 按 'p' 暂停/继续
   - 按 'i' 记录干扰
   - 按 'q' 放弃当前番茄钟
4. 完成后自动进入休息时间
5. 查看统计、成就和历史记录

## 数据文件

程序会在运行目录下创建以下数据文件：
- `pomodoro_data.txt` - 番茄钟记录
- `achievements_data.txt` - 成就数据
- `todolist_data.txt` - 待办事项
- `password_dic.txt` - 密码数据
- `diary_dic.txt` - 日记数据
- `notepad_dic.txt` - 记事本数据
